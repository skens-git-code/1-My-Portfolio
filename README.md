## ⚡ **Installation & Setup**

### 📦 **Prerequisites**

- Modern browser (Chrome 90+, Firefox 88+, Safari 15+, Edge 90+)
- JavaScript enabled
- WebGL support (optional, for 3D effects)

### 🚀 **Quick Start**

```bash
# Clone the repository
git clone https://github.com/skens-git-code/portfolio.git

# Navigate to project
cd portfolio

# No build step required!
# Open index.html in your browser
# Or serve with any static server

# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```

### 🔧 **Configuration**

```javascript
// config.js - Optional customization
const CONFIG = {
  loading: {
    minDisplayTime: 800,    // Minimum loading screen duration
    maxDisplayTime: 4000,   // Force hide timeout
    progressStep: 20        // Progress increment speed
  },
  animations: {
    scrollThrottle: 16,     // 60fps scroll events
    intersectionThreshold: 0.08, // Section detection
    staggerDelay: 80        // Animation delay between items
  },
  particles: {
    count: {
      desktop: 40,          // High-performance particles
      mobile: 20            // Battery conscious
    }
  }
};
```
